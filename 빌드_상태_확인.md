# 빌드 상태 확인 및 다음 단계

## 현재 상황

Bubblewrap 빌드가 아직 완료되지 않았거나 진행 중일 수 있습니다.

## 확인 방법

### 1. 빌드가 실행 중인지 확인

터미널에서 다음을 확인하세요:
- 빌드 진행 메시지가 보이는지
- "Building..." 또는 "Gradle build..." 같은 메시지가 있는지

### 2. 빌드 완료 확인

빌드가 완료되면 다음 파일들이 생성됩니다:
- `app-release-bundle.aab` (Play Store 업로드용)
- `app-release-signed.apk` (테스트용)
- `assetlinks.json` (SHA-256 지문 포함)

## 다음 단계

### 옵션 1: 빌드 다시 실행

터미널에서:
```powershell
cd nexsupply-twa
bubblewrap build
```

비밀번호를 묻으면: `nexsupply2024`

### 옵션 2: 빌드 완료 대기

빌드가 진행 중이라면 완료될 때까지 기다리세요 (보통 5-10분 소요)

### 옵션 3: 수동으로 SHA-256 지문 추출

빌드가 완료되었지만 assetlinks.json이 없다면:

1. 키스토어에서 직접 추출:
```powershell
cd nexsupply-twa
keytool -list -v -keystore android.keystore -alias twa
```

비밀번호: `nexsupply2024`

2. 출력에서 "SHA256:" 또는 "SHA-256:" 찾기

## 빌드 완료 후

빌드가 완료되면:
1. `assetlinks.json` 파일에서 SHA-256 지문 추출
2. 프로젝트의 `.well-known/assetlinks.json` 업데이트
3. Git 커밋 및 푸시
